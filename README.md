# Pomood Privacy Policy

This repository hosts the privacy policy for the Pomood mobile application.

## Privacy Policy URL

Once GitHub Pages is enabled, the privacy policy will be available at:

```
https://YOUR_USERNAME.github.io/pomood-privacy-policy/
```

## Setup Instructions

1. **Enable GitHub Pages:**
   - Go to repository Settings â†’ Pages
   - Source: Deploy from a branch
   - Branch: `main`
   - Folder: `/ (root)`
   - Click Save

2. **Wait 1-2 minutes** for GitHub Pages to deploy

3. **Access your privacy policy:**
   - Visit: `https://YOUR_USERNAME.github.io/pomood-privacy-policy/`

## Update Contact Email

Before using in Google Play Console, edit `index.html` and replace `[YOUR_EMAIL_HERE]` with your actual contact email.
